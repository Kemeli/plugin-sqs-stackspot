Descrição

Plugin para manipular mensagens em uma fila SQS

Requirements

  • Python 3.8 ou superior.
  • Chalice 1.25.1 ou superior.
  • Boto3 1.18.44 ou superior.

Sobre

O SQS Library é um plugin que fornece uma biblioteca para interagir com o serviço Amazon Simple Queue Service (SQS). O plugin é desenvolvido usando a biblioteca boto3. Ele oferece funções convenientes para enviar, receber e manipular mensagens em uma fila SQS.

Implementação

A implementação do SQS Library é feita da seguinte forma:

  • Importação das bibliotecas necessárias:
    • 		
			from chalice import Chalice
			import boto3
			import uuid
  • Definição das constantes, que incluem as URLs de endpoint, os detalhes da fila e a região:
    • 		
			ENDPOINT_URL_SEND = 'http://host.docker.internal:4566'
			ENDPOINT_URL_RECEIVE = "http://host.docker.internal:4566/_aws/sqs/messages"
			QUEUE_URL = "http://queue.localhost.localstack.cloud:4566/000000000000"
			QUEUE_NAME = 'MinhaFila.fifo'
			REGION = 'us-east-1'
  • Criação do cliente SQS com o endpoint e região especificados:
    • 		
			sqs_client = boto3.client('sqs', endpoint_url=ENDPOINT_URL_SEND, region_name=REGION)
  • Implementação das funções da biblioteca, como enviar mensagens FIFO, enviar mensagens padrão, receber mensagens e manipular informações das mensagens. Cada função possui uma descrição detalhada de seus parâmetros e retorno.

Utilização

Para utilizar o SQS Library como um plugin em seu projeto, siga as etapas abaixo:

  • Certifique-se de ter instalado a biblioteca Boto3 em seu ambiente Python.
  • Copie o código do SQS Library e cole-o em seu arquivo Python ou ambiente de desenvolvimento.
  • Defina as constantes ENDPOINT_URL_SEND, ENDPOINT_URL_RECEIVE, QUEUE_URL, QUEUE_NAME e REGION de acordo com sua configuração da fila SQS.
  • Utilize as funções fornecidas pela biblioteca para enviar, receber e manipular mensagens em sua fila SQS.
